<task type="auto">
  <name>Task 2: Create tree layout algorithm for mind maps</name>
  <files>live-canvas-mcp/src/layouts/tree.ts</files>
  <action>
Create a pure function module for tree layout positioning. This calculates x/y positions for nodes in a tree structure.

Export interfaces:
```typescript
interface TreeNode {
  id: string;
  label: string;
  children?: TreeNode[];
}

interface PositionedNode {
  id: string;
  label: string;
  x: number;
  y: number;
  width: number;
  height: number;
  parentId?: string;
}

interface LayoutResult {
  nodes: PositionedNode[];
  connections: Array<{ fromId: string; toId: string }>;
}

interface LayoutOptions {
  startX?: number;      // Default 400
  startY?: number;      // Default 300
  levelGap?: number;    // Vertical gap between levels, default 120
  siblingGap?: number;  // Horizontal gap between siblings, default 40
  nodeWidth?: number;   // Default 160
  nodeHeight?: number;  // Default 60
}
```

Export function:
```typescript
function layoutTree(root: TreeNode, options?: LayoutOptions): LayoutResult
```

Algorithm (simplified Reingold-Tilford style):
1. Calculate subtree widths recursively (sum of children widths + gaps)
2. Position root at startX, startY
3. Position children centered below parent, spaced by siblingGap
4. Recurse for each child's subtree
5. Collect all positioned nodes and their parent connections

Keep it simple - this is for <50 nodes. Don't optimize prematurely.

   - Creates arrow skeletons for each connection
   - Broadcasts: `{ type: "diagram_elements", diagramType: "mindmap", elements: [...skeletons] }`
   - Returns success message with node count
