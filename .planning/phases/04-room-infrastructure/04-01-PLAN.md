<task type="auto">
  <name>Task 3: Create Socket.IO server with room event handlers</name>
  <files>
    live-canvas-mcp/src/server/socketio.ts
  </files>
  <action>
Create `src/server/socketio.ts`:

1. Imports:
   - `Server as SocketIOServer` from "socket.io"
   - `Server as HttpServer` from "http"
   - Types from "../rooms/types.js"
   - `createRoomManager` from "../rooms/manager.js"

2. Module state:
   - `let io: SocketIOServer | null = null`
   - `const roomManager = createRoomManager()`

3. Export `initSocketIO(httpServer: HttpServer, getCanvasState: () => any)`:
   - Create Socket.IO server attached to httpServer with CORS config (origin: "*")
   - Type it with `<ClientToServerEvents, ServerToClientEvents, {}, SocketData>`

4. Connection handler (`io.on("connection", socket => {...})`):

   **create_session handler:**
   - Call `roomManager.createRoom(socket.id, getCanvasState())`
   - Join the Socket.IO room: `socket.join(room.code)`
   - Set `socket.data = { roomCode: room.code, isHost: true }`
   - Generate URL: `http://localhost:${PORT}/join/${room.code}`
   - Callback with `{ code: room.code, url }`
   - Log: `[Room] Created: ${room.code} by ${socket.id}`

   **join_session handler (code, callback):**
   - Normalize: `code.toUpperCase()`
   - Call `roomManager.joinRoom(code, socket.id)`
   - If success:
     - Join Socket.IO room: `socket.join(code)`
     - Set `socket.data = { roomCode: code, isHost: false }`
     - Notify others: `socket.to(code).emit("user_joined", { socketId: socket.id })`
     - Send canvas state: `socket.emit("canvas_state", room.canvasState)`
     - Callback `{ success: true }`
   - If fail: callback `{ success: false, error: "Session not found" }`
   - Log: `[Room] Joined: ${code} by ${socket.id}`

   **leave_session handler:**
   - Call `roomManager.leaveRoom(socket.id)`
   - If was in room: `socket.leave(roomCode)`
   - Notify room: `io.to(roomCode).emit("user_left", { socketId: socket.id })`
   - Clear socket.data

5. Export `getIO(): SocketIOServer | null` for other modules to broadcast.

IMPORTANT: Do NOT wire into index.ts yet - that's Plan 02's job. This file stands alone.
  </action>
